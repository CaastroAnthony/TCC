Sobre o Curso

O curso técnico em Desenvolvimento de Sistemas é a porta de entrada para o universo da programação e da tecnologia. Com um enfoque prático e alinhado às necessidades do mercado, você aprenderá a construir sistemas robustos e modernos.

O que você vai dominar:

Programação Frontend: HTML, CSS, JavaScript e frameworks modernos como React e Vue.js;

Desenvolvimento Backend: Criação de APIs, integrações e servidores com Node.js, Python (Django/Flask), Java e PHP;

Desenvolvimento Mobile: Construção de apps nativos e híbridos para Android e iOS com React Native, Flutter ou Kotlin;

Banco de Dados: Modelagem, SQL (MySQL, PostgreSQL) e NoSQL (MongoDB), além de otimização de queries;

Controle de Versão: Uso profissional do Git e GitHub;

Metodologias Ágeis: Scrum, Kanban e gestão de projetos;

Segurança da Informação: Princípios básicos para proteger aplicações;

Deploy e DevOps: Introdução a Docker, CI/CD e cloud (AWS, Azure)..

Conteúdo abordado no curso

  • Programação Frontend
  • Programação Backend
  • programação Mobile
  • Modelagem e Desenvolvimento de Bancos de Dados
