Sobre o Curso de DS

O curso de Desenvolvimento de Sistemas prepara profissionais para atuar no desenvolvimento de programas, aplicativos e sistemas computacionais. Durante o curso, você aprenderá:

  • Lógica de programação e algoritmos
  • Desenvolvimento web e mobile
  • Banco de dados e estrutura de dados
  • Engenharia de software e metodologias ágeis
  • E muito mais!

Disciplinas do Curso

1º Semestre

Lógica de Programação

Introdução aos conceitos fundamentais de programação, algoritmos e estruturas de controle.

  • Variáveis e tipos de dados
  • Estruturas condicionais
  • Laços de repetição
1º Semestre

Fundamentos de TI

Conceitos básicos de hardware, software e sistemas operacionais.

  • Arquitetura de computadores
  • Sistemas operacionais
  • Redes de computadores
2º Semestre

Banco de Dados

Introdução aos sistemas de gerenciamento de bancos de dados relacionais.

  • Modelo relacional
  • SQL básico e avançado
  • Normalização de dados

Demonstrações Práticas

Editor de Código Interativo

// Exemplo de código JavaScript
function calcularMedia(notas) {
    let soma = 0;
    for (let nota of notas) {
        soma += nota;
    }
    return soma / notas.length;
}

const notas = [7.5, 8.0, 6.5, 9.0];
console.log("Média:", calcularMedia(notas));
