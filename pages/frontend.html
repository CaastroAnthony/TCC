Front-End

O que é Front-End?

O Front-End é a parte visível e interativa de um site ou aplicativo. Tudo o que o usuário vê na tela — textos, imagens, botões e menus — faz parte do Front-End. Ele representa o lado do usuário, ou seja, o que é mostrado e com o que se pode interagir.

Principais tecnologias:

  • HTML: define a estrutura da página (títulos, parágrafos, imagens, links).
  • CSS: define a aparência da página (cores, fontes, espaçamento, layout).
  • JavaScript: adiciona interatividade (botões, formulários, animações).

Exemplo simples: em uma página de login, o HTML cria o formulário, o CSS deixa os campos organizados e bonitos, e o JavaScript verifica se o usuário digitou os dados antes de enviar.

Resumo : Front-End = tudo o que o usuário vê e interage no site. HTML = estrutura, CSS = estilo, JavaScript = comportamento.

Comandos Básicos de Front-End

HTML

Comando / TagPara que serve
<html>Define o início de um documento HTML.
<head>Contém metadados (título, links, scripts).
<body>Onde fica o conteúdo visível da página.
<h1> a <h6>Títulos (h1 é o maior, h6 o menor).
<p>Define um parágrafo de texto.
<a href="">Cria um link clicável.
<img src="">Insere uma imagem.
<div>Container genérico para agrupar elementos.
<span>Usado para estilizar partes de texto.
<ul>, <ol>, <li>Listas: não ordenada, ordenada, e itens de lista.
<input>Campo de entrada (texto, email, etc).
<button>Cria um botão.
<form>Agrupa campos de formulário para envio.
<br>Quebra de linha.

CSS (estilização)

PropriedadePara que serve
colorDefine a cor do texto.
background-colorCor de fundo.
font-sizeTamanho da fonte.
text-alignAlinhamento do texto.
marginEspaço externo ao redor do elemento.
paddingEspaço interno do elemento.
borderAdiciona borda ao elemento.
displayDefine o tipo de exibição (block, flex, etc).
flexUsado para criar layouts responsivos.
positionPosicionamento (relative, absolute, etc).
width / heightLargura e altura do elemento.
:hoverAplica estilo ao passar o mouse.
@mediaEstilização responsiva para telas diferentes.
