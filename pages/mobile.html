Desenvolvimento Mobile

O que é Desenvolvimento Mobile?

O Desenvolvimento Mobile é a criação de aplicativos para dispositivos móveis como smartphones e tablets. Ele pode ser feito de forma nativa (linguagens específicas para cada sistema operacional) ou híbrida (um único código que roda em Android e iOS).

Principais tecnologias:

  • Android: desenvolvido principalmente em Kotlin ou Java.
  • iOS: desenvolvido com Swift.
  • React Native: cria apps híbridos usando JavaScript.
  • Flutter: framework do Google que usa a linguagem Dart.

Resumo : Mobile = aplicativos que rodam no celular. Nativo = mais performance, Híbrido = código único para várias plataformas.

Comandos e Conceitos Básicos de Mobile

React Native (JavaScript)

ComandoPara que serve
npx react-native init appCria um novo projeto React Native.
npm installInstala as dependências do projeto.
npx react-native run-androidExecuta o app no Android.
npx react-native run-iosExecuta o app no iOS.
import { Text, View } from "react-native"Importa componentes básicos.

Flutter (Dart)

ComandoPara que serve
flutter create appCria um novo projeto Flutter.
flutter runExecuta o app no emulador/dispositivo.
MaterialApp()Define a estrutura principal do app.
Scaffold()Cria uma tela padrão com barra superior e corpo.
setState()Atualiza a interface quando os dados mudam.

Kotlin (Android Nativo)

ComandoPara que serve
class MainActivity : AppCompatActivity()Classe principal da aplicação Android.
setContentView(R.layout.activity_main)Define o layout da tela.
findViewById<Button>(R.id.botao)Busca um elemento da tela pelo ID.
button.setOnClickListener { ... }Define ação ao clicar em um botão.
Toast.makeText(...)Exibe uma mensagem rápida na tela.
