Banco de Dados

O que é Banco de Dados?

Um Banco de Dados é um sistema organizado para armazenar, gerenciar e recuperar informações de forma eficiente. Pode ser relacional (tabelas com relações) ou não relacional (NoSQL).

Principais tipos de banco de dados:

  • MySQL: Sistema de banco de dados relacional amplamente usado.
  • PostgreSQL: Relacional avançado, com suporte a recursos complexos.
  • MongoDB: Banco NoSQL orientado a documentos.
  • SQLite: Banco de dados leve, usado em apps móveis.

Resumo : BD Relacional = tabelas, SQL; BD Não Relacional = documentos, chaves e valores.

Comandos e Conceitos Básicos de Banco de Dados

SQL (MySQL/PostgreSQL)

ComandoPara que serve
CREATE DATABASE nome;Cria um novo banco de dados.
CREATE TABLE tabela (...);Cria uma nova tabela.
INSERT INTO tabela VALUES (...);Insere dados na tabela.
SELECT * FROM tabela;Consulta todos os dados da tabela.
UPDATE tabela SET coluna=valor;Atualiza valores de uma tabela.

MongoDB (NoSQL)

ComandoPara que serve
use nomeDBSeleciona ou cria um banco de dados.
db.createCollection("nome")Cria uma coleção.
db.nome.insertOne({campo: valor})Insere um documento na coleção.
db.nome.find()Consulta documentos da coleção.
db.nome.updateOne(...)Atualiza documentos.
